Variáveis

Em JS, além de utilizar a palavra "var", também podemos utilizar a palavra "let" para criar variáveis

"Armazenar dados na memória do computador"

Identificadores

  • Podem começar com letra, $ ou _
  • Não podem começar com números
  • É possível usar letras ou números
  • É possível usar acentos e símbolos
  • Não podem conter espaços
  • Não podem ser palavras reservadas (palavras que exitem nos códigos JS)
  • Maiúsculas e Minúsculas fazem diferença

Tipos Primitivos

"typeof" - mostrar o tipo da variável

  • Number
    • reais - float
    • inteiros
    • NaN - Not a Number
    • Infinity
  • String
    • Cadeias de caracteres
    • Pode ser números
    • Delimitados por " " , ' ' ou ` `
  • Boolean
    • True
    • False
  • Null
  • Undefined
  • Object
    • Array - Vetores
  • Function
