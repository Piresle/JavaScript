Operadores

Aritméticos

+ adição ou concatenação
- subtração lo
* multiplicação
/ divisão real (float)
% divisão inteira (int)
** exponenciação

Precedência

1° ( )
2° **
3° * / %
4° + -

Atribuição

= "recebe"
+= o mesmo que "var = var + numero" (-=, *=, ...)
++ incremento (var++ é o mesmo que var = var + numero)

Relacionais

> maior que
< menor que
>= maior ou igual a
<= menor ou igual a
== igual a
!= diferente de
=== operador de identidade ("idêntico")

* O resultado é sempre boolean (True / False)

* Não tem ordem de Precedência

Lógicos

! negação (NOT)
&& conjunção (AND)
|| disjunção (OR)

Precedência

1° !
2° &&
3° ||
