Conversão

String -> Número

Number.parseInt(n) -> converte a string n para um número inteiro

Number.parseFloat(n) -> converte a string n para um número decimal (float)

OU

Number(n) -> converte a string para um valor númerico (float ou int, qualquer um dos dois)

Número -> String

String(n) -> converte a variável n para string

n.toStrig() -> converte a variável n para string
