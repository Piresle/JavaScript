Condições

"desvios condicionais" em um fluxo de código

Estrutura de controle

  • If (condição) {
    Código
    }
    else {
    código
    }
    • "se"

Tipos de Condição

  • Simples
    • só tem um bloco
    • se a condição for True, executa o código normalmente; se for False, nada vai acontecer em especial
  • Composta
    • mais de um bloco de código
    • caso a condição seja verdadeira, executa o bloco True (if); caso seja falsa, executa o bloco False (else)
