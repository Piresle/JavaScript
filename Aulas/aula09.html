Introdução ao DOM - Aula 09

DOM - Document Object Model

"conjunto de objetos no navegador que dá acesso aos componentes internos do seu website."

Árvore DOM

  • window - "raiz" da árvore DOM (janela do navegador)
    • location - localização do site (URL, página atual, página anterior)
    • document - "documento atual"
      • HTML - estrutura do site
        • head
          • meta
          • title
          • ...
        • body
          • h1
          • p
          • div
          • ...
    • history - histórico

Seleção

  • por Marca
    • getElementsByTagName()
      • dentro de "document"
      • seleção por tag
      • document.getElementsByTagName(tag)
  • por ID
    • getElementById()
  • por Nome
    • getElementsByName()
    • através do atributo "name" em uma tag
  • por Classe
    • getElementsBtClassName()
  • por Seletor
    • querySelector()
      • ( tag + seletor(# ou .) + atributo)
      • mais prático para alterações rápidas
    • querySelectorAll()
